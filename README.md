# Kazcom

## Zainstalowanie zależności oraz uruchomienie aplikacji
```
cd chat_vue
yarn install
node server/main.js
```

### Technologie użyte w projekcie:
- express.js
- vue3
- socket.io
- axios.js
- MongoDB

## Oprogramowanie na którym aplikacja była uruchamiana:
- Node.js v.18.14.0
